Ubuntu 14.04 安装 Composer+Laravel 配置

zhonger 2016-02-05 17:29:05 +0900
本文总共 333 字 · 阅读全文大约需要 1 分钟

Linux 下配置 laravel 环境

安装前提：系统已安装好 php 环境，可在全局使用 php 命令。

安装 compsoer 并设置为全局调用

curl -sS https://getcomposer.org/installer | php
mv composer.phar /usr/local/bin/composer

使用 composer 建立 laravel installer 安装工具

composer global require "laravel/installer"

将 laravel 命令添加进用户环境变量

  打开~/.bashrc文件最末添加命令：PATH=$PATH:~/.composer/vendor/bin source ~/.bashrc 命令使环境变量变化生效。

完成配置

  可以在全局使用laravel new {应用名称}了。
