重装系统

添加新用户及赋权

新增用户及配置默认密码

sudo useradd 用户名
sudo passwd 用户名

添加用户到 sudo 组

sudo chmod +w /etc/sudoers
sudo vi  /etc/sudoers 
# 加入 用户名 ALL=(ALL:ALL) ALL

配置默认 Shell

  更改Linux 用户的 Shell 的操作方法如下所示：

# 查看当前用户的 shell 方式 
echo $SHELL    
>> /bin/sh

# 更换 shell 操作方式为 `/bin/bash`   
sudo vim /etc/passwd  
# 在用户行尾加上`/bin/bash`

# 退出系统再次登录
